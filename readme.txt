This plugin allows you to change the absolute url to relative url format.  

== Description ==

File Relative Path is a simple Wordpress plugin but powerful in SEO Enhancement, this plugin allows user to override the default/absolute url to relative url format. 

NOTE: Please always read the changelog for the plugin updates.

**Plugin Options**

**Basic**
  * Images Become Relative is the default function ( Once the plugin is activated its automatically change the absolute url to relative url format)

== Frequently Asked Questions ==

= What will happen when I remove this plugin? =

When you remove the plugin the data save in the database after the plugin is de-activated it's still there.
